CONTEXTE : POURQUOI L'ASYNCHRONISME ?

exemple code : 

server.get('/users', async (_req, res) => {
    try {
      const result = await db.query('SELECT * FROM users');
      res.send(result.rows);
    } catch (err) {
      console.error('Erreur...', err);
      res.status(500).send({ error: '...' });
    }
  });

Ce code doit parfois attendre des choses lentes :
  * Récupérer des données d'une base de données.
  * Appeler une API externe.
  * Lire un fichier.
  * etc.

Or, JavaScript est monothread -> s'il attend bloqué, tout s'arrête. C'est pourquoi on utilise l'asynchronisme.
