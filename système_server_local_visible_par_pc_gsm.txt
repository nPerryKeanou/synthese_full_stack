Le server Express tourne sur le pc (ex: http://localhost:3000).
    * problème : sur ton smartphone, localhost n'existe pas -> "localhost" = l'appareil lui-même.
    * Solution : utiliser l'adresse IP Local de ton pc (par ex: 192.xxx.x.xx:3000).

Étapes : 
    1. Vérifie ton IP local sur ton PC;
        linux/max: ifconig ou ip a
    2. Dans ton front-end (react native), quand tu fais des requêtes API, utilise :
        const API_URL = "http://192.xxx.x.xx:3000";

        attention le pc et le gsm doivent être sur le même wi-fi.
    3. Sur ton serveur, Express, assure-toi que tu écoutes sur 0.0.0.0 :
        app.listen(3000, "0.0.0.0", () => {
            console.log("Serveur dispo sur le réseau local! ");
        })
    -> ça permet d'accepter les connexions venant d'autres appareils du réseau.
