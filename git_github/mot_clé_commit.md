| Mot-clé    | Explication                                                                                  | Exemple de message                                        |
|------------|---------------------------------------------------------------------------------------------|----------------------------------------------------------|
| **feat**   | Ajout d’une nouvelle fonctionnalité                                                         | `feat: add user authentication with JWT`                 |
| **fix**    | Correction d’un bug                                                                         | `fix: resolve crash when user clicks on submit button`   |
| **docs**   | Modification ou ajout de documentation                                                     | `docs: update README with setup instructions`            |
| **style**  | Changements liés à la mise en forme (formatage, espaces, etc.) sans impact fonctionnel      | `style: fix indentation in login component`              |
| **refactor**| Modification du code sans ajout de fonctionnalité ni correction de bug (amélioration de la structure) | `refactor: simplify user service logic`                   |
| **perf**   | Amélioration des performances                                                               | `perf: optimize database queries for faster load`        |
| **test**   | Ajout ou modification des tests                                                             | `test: add unit tests for payment module`                 |
| **chore**  | Tâches diverses (build, outils, dépendances, scripts) sans impact sur le code métier        | `chore: update dependencies to latest versions`          |
| **build**  | Modifications liées au système de build ou aux dépendances externes                         | `build: configure webpack for production`                 |
| **ci**     | Changements liés à l’intégration continue (CI)                                             | `ci: add GitHub Actions workflow for tests`               |
| **revert** | Annulation d’un commit précédent                                                           | `revert: fix: remove user logout bug`                     |
| **wip**    | Travail en cours (work in progress)
